# pyueye

This is basic GUI of uEye.  
Run ueye_main.py, you can look live-image.  
Also, this code can save raw-image.  

The program inspired from exmaple of IDS.  
https://en.ids-imaging.com/techtipps-detail/en_techtip-embedded-vision-kit.html

## Dependencies
This program depends on the following libraries:

numpy  
pyueye  
opencv-python  
pyqt4  
Also, this code should be compatible with Python versions 3.5-3.7.
